1.    Assume the following has been typed in:
        Scanner scan = new Scanner(System.in);
      Which line of code declares an int x and sets it equal to input from the user?
Ans:  int x = scan.nextInt();

2.    Which line of code declares a double variable x and sets it equal to 5.0?
Ans:  double x = 5.0;

3.    Is an int bigger or smaller in memory than a double?
Ans:  smaller
